# Form README

This repository contains the form implementation for the tutorial.

## Overview

The form collects user input and validates values before submission.

## Files

- `index.html` - form markup
- `styles.css` - form styling
- `script.js` - validation and submission logic

## Usage

1. Open the form page in your browser.
2. Fill in the required fields.
3. Submit the form and check for validation feedback.

## Notes

- Make sure all required fields are completed.
- Review the browser console for any validation messages.
